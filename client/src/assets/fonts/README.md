# Cairo Font for PDF/PPT Export

Cairo is used for RTL Arabic text in slide exports.

**Source**: [@fontsource/cairo](https://www.npmjs.com/package/@fontsource/cairo) (installed via pnpm)

Font files are loaded via the fontLoader utility which imports from `@fontsource/cairo/files/`.

**Variants**:
- Regular (400)
- SemiBold (600)
- Bold (700)
